# Projeto Python IA: Artificial Intelligence

### Case: Detect if a message it's a Spam or not.

In this project were explored data predict models, using librarys such Pandas and Scikit-learn, The two ways to see the efficiency of the AI tools were: KNN and Decision Tree.
The data explored were random mail boxes found at: https://github.com/20Candy/SPAM-HAM-classification/blob/main/data.csv.
